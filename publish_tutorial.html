Publish Webpage Tutorial

For this webpage, I will show you how to upload your webpage on Icarus Web, and instruction how to compose your URL, and giving you some example my URL link

Four Step

Step 1: Download FTP tool:

Cyberduck

Cyberduck can help you drag and drop from your computer onto Icarus. Below is an instruction for get in Cyberduck

Download and install Cyberduck by going to this link above. Once installed, run Cyberduck. Click "File" and choose "Open Connection" or Ctrl + O for opening

Step 2: Getting information needed for entered

Server:

icarus.cs.weber.edu or 137.190.19.20

Port:

22

Username:

Initials then the last five digits of W number

For Example:

  • Name: Qui Ngo
  • W#: 01413456
  • Username: qn13456
Password:

First name (first letter capitalized) followed by "cs!"

  • Name: Qui Ngo
  • Password: Quics!
Note: Make sure to choose Make sure to choose SFTP

Step 3: Upload file

Open the public_html directory. Files can be dragged single file for single file or keep shift and click mouse to and from your computer, and drop in public_html file. Or, you can click on file and choose uploap file

Step 4: Compose your URL

After you finish three steps above that you published your webpage on web already. Your URL will compose like this:

https://icarus.cs.weber.edu/ ~your username/your folder that store html file/name html file.html
For example:

https://icarus.cs.weber.edu/~qn13120/HW1Covid/covid.html

https://icarus.cs.weber.edu/~qn13120/

https://icarus.cs.weber.edu/~qn13120/distractions.html
